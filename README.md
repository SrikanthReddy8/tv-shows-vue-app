# 🎬 TV Shows Dashboard

A Vue 3 application that displays TV shows using the TVMaze API.
It includes search functionality, genre-based browsing, and a smooth horizontal scrolling UI.

---

## 🚀 Features

* 🔍 Search TV shows
* 🎭 Browse shows by genre
* ⭐ View popular shows
* 🎬 Horizontal carousel with navigation
* ⚡ Fast state management using Pinia

---

## 📦 Prerequisites

Before running the project, make sure you have:

* **Node.js** (v16 or above) → https://nodejs.org/en
* **npm** (comes with Node.js)

Verify installation:

```sh
node -v
npm -v
```

---

## ⚙️ Setup

Clone the repository:

```sh
git clone https://github.com/SrikanthReddy8/tv-shows-vue-app.git
cd tv-shows-vue-app
```

Install dependencies:

```sh
npm install
```

---

## ▶️ Development

Run the app in development mode:

```sh
npm run dev
```

Open in browser:

```
http://localhost:5173
```

---

## 🏗️ Build

Create a production build:

```sh
npm run build
```

---

## 🧪 Testing

Run unit tests using Vitest:

```sh
npm run test:unit
```

---

## 🛠️ Tech Stack

* Vue 3 (Composition API)
* Pinia (State Management)
* Vue Router

* TVMaze API
